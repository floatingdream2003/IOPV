# 基金实时估值

一个基于 Next.js 的基金实时估值应用，支持添加基金代码查看实时估值、涨跌幅和前10重仓股票信息。
预览地址: <https://floatingdream2003.github.io/IOPV/>

## 功能特性

-  **实时估值** - 自动刷新基金估值数据，支持自定义刷新频率
-  **重仓股票** - 显示基金前10重仓股票及其涨跌幅
-  **响应式设计** - 支持桌面端和移动端，提供卡片和表格两种视图
-  **分组管理** - 支持持有基金标记和自定义分组功能
-  **本地存储** - 数据本地保存，刷新页面不丢失
-  **现代UI** - 毛玻璃效果，深色主题，流畅动画

## 技术栈

- **框架**: Next.js 14
- **UI库**: React 18
- **动画**: Framer Motion
- **样式**: CSS Variables + 响应式设计
- **数据源**: 腾讯财经，东方财富API

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看应用。

### 构建生产版本

```bash
npm run build
```

## 使用说明

1. **添加基金**: 输入6位基金代码（如：004433）
2. **选择类型**: 可标记为"持有基金"或添加到"自建分组"
3. **查看数据**: 实时估值、涨跌幅、重仓股票信息
4. **管理基金**: 支持删除、分组、收藏等操作
5. **自定义设置**: 调整刷新频率（最低5秒）
